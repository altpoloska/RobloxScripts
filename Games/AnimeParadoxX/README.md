# Anime Paradox X Macro Recorder — adapter v3.23

Адаптация модульного UTDZ-рекордера под Anime Paradox X с `UnitAction:FireServer(...)`.

## Сборка через luabundler

Выполните из корня проекта:

```powershell
luabundler bundle "src/Main.lua" -p "?.lua" -o "AnimeParadox.lua"
```

Все импорты содержат полный префикс `src`, например `require("src.UI.Interface")`. Поэтому шаблон `?.lua` разрешает этот импорт непосредственно в `src/UI/Interface.lua` и не зависит от особенностей обработки относительного package path в разных версиях luabundler.

## Реализовано

- По умолчанию используется **x1**. Скорость **x2** включается только тогглером `Auto 2x speed` на вкладке Main и отправляется через `ReplicatedStorage.Remotes.GameSpeed`.
- Place: `UnitAction:FireServer("Place", inventoryUuid, Vector3, rotation)`.
- Upgrade: `UnitAction:FireServer("Upgrade", entity, false)`.
- Sell: `UnitAction:FireServer("Sell", entity)`.
- Replay: `ReplicatedStorage.Remotes.StageEnd:FireServer("Replay")`.
- Запись и воспроизведение макросов с метками `Unit name - N`.

## Как решён UUID

При записи адаптер получает имя из атрибута `UnitName` у записи:

`PlayerGui.UnitInventory.Inventory.Content.Units.UnitsListFrame.List[uuid]`

Экипировка конкретной копии проверяется через `List[uuid].Frame.Checkmark.Visible`. UUID используется только во время текущей игровой сессии и **не сохраняется в макрос**, поскольку он различается у игроков. В макрос записываются имя юнита, позиция, rotation и стабильная метка размещённого экземпляра. При воспроизведении UUID заново находится среди экипированных записей по атрибуту `UnitName`, поэтому макрос можно передавать другому игроку. Перед Place адаптер запоминает подходящие модели, у которых уже есть `HumanoidRootPart`. После ремоута он ждёт, у какой модели с именем вроде `Ichigo_0.4` появился `HumanoidRootPart`, и связывает её с заранее зарезервированной меткой `Ichigo - 1/2/3…`. Upgrade/Sell затем используют актуальный Instance.

## Индикатор волны

Волна читается по точному пути `PlayerGui.GameHUD.WaveFrame.WaveInfo.Waves.CurrentWave.Text`; из текста извлекается первое число.
